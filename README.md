# ISP_DZ2 (Unbreakable stack)

This stack implementation will never break. Probably. 

## Build && run
On linux, you can build using CMake
    
    cmake -S ./ -B out && cmake --build out
    
Run generated executable (filepath is optional argument)
   
    out/executable   

Run unit tests
   
    out/tests
